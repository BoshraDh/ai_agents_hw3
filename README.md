## Project Overview

This project implements a **fully autonomous, multi-agent pipeline** using the **CrewAI** framework to produce a 15-page academic paper comparing two leading AI orchestration frameworks: **LangGraph** and **CrewAI**.

Three specialised AI agents — a Researcher, a Writer, and a Reviewer — collaborate sequentially with no human intervention in the content generation process. The pipeline demonstrates real-world multi-agent orchestration: each agent receives the previous agent's output, processes it according to its defined role and goal, and passes the result downstream.

> **Note:** This academic paper and its supporting content (research notes, paper draft, bibliography, figures, and peer review) were **generated entirely by autonomous AI agents** (Researcher, Writer, Reviewer) **orchestrated via CrewAI**. The agents used `gpt-4o` as the underlying language model.

## How to Run the Project

### Prerequisites

- Python 3.10 or higher
- An OpenAI API key with access to `gpt-4o`
- (Optional, for PDF recompilation) MiKTeX or TeX Live with `pdflatex`

### Step 1 — Clone the repository

```bash
git clone https://github.com/BoshraDh/ai_agents_hw3.git
cd ai_agents_hw3
```

### Step 2 — Create a virtual environment

```bash
# Windows
python -m venv .venv
.venv\Scripts\activate

# macOS / Linux
python3 -m venv .venv
source .venv/bin/activate
```

### Step 3 — Install dependencies

```bash
pip install -r requirements.txt
```

### Step 4 — Configure the API key

```bash
# Windows
copy .env.example .env

# macOS / Linux
cp .env.example .env
```

Open `.env` and set your key:

```
OPENAI_API_KEY=sk-...your-key-here...
```

> `.env` is listed in `.gitignore` and is never committed to the repository.

### Step 5 — Run the agent pipeline

```bash
python src/main.py
```

The three agents run sequentially (Researcher → Writer → Reviewer) and write all outputs to `output/`. Runtime: approximately 3–8 minutes depending on API response times.

### Step 6 — (Optional) Regenerate figures

```bash
python src/generate_images.py
```

Regenerates `latex/images/arch_comparison.png` and `latex/images/performance_chart.png` using matplotlib. No API key required.

### Step 7 — (Optional) Recompile the PDF

```bash
cd latex
pdflatex paper.tex
bibtex paper
pdflatex paper.tex
pdflatex paper.tex
```

> The compiled `latex/paper.pdf` (15 pages, 514 KB) is already committed to the repository.
